# Doctor Appointment System

## 📌 Description
This is a full-stack web application built using React, Node.js, Express, and MongoDB.

## 🚀 Features
- User Registration & Login
- Book Appointment with Doctors
- View My Bookings
- Cancel Appointment
- Admin Dashboard

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## ▶️ How to Run
1. Clone the repo
2. Run backend:
   npm install
   node server.js

3. Run frontend:
   cd client
   npm install
   npm start
